  • 【15/01/07 - F.C.新聞報】El Patron Alberto 再奪腰帶 *

2015年1月7日 星期三

2015年1月7號

在波多黎各當地，隸屬於 AAA ( Asistencia Asesoría y Administración ) 的 El Patrón Alberto (小名鑑) 又再奪下一條冠軍腰帶，他參戰 WWL ( World Wrestling League ) 於一月六號歡慶主顯節的大會 Idols of Wrestling 上，在三方賽擊敗了前王者 Glamour Boy Shane，拿下 WWL 世界重量級冠軍腰帶 ( WWL World Heavyweight Championship )。由於 Alberto 已經在一個月前的 AAA 大會 Guerra de Titanes 上攻下 AAA 的世界冠軍王座，使得他在現在成為了雙冠王。這場三方賽的另一個挑戰者為 El Mesías。

Alberto 目前已經創下七次的世界冠軍紀錄，其中四次是在 WWE ( World Wrestling Entertainment ) 拿下。另外就是在 AAA 與 CMLL ( Consejo Mundial de Lucha Libre ) 攻下，以及這次的 WWL 腰帶。他是目前歷史上第一位曾拿過墨西哥兩大團 AAA 與 CMLL 世界冠軍腰帶的選手。
